# TON Airdrop Hunter AI 🤖

TON Airdrop Hunter AI is a Telegram bot and web tool that helps users discover, track, and participate in airdrops and reward opportunities across the TON ecosystem.

The bot scans for new campaigns and notifies users about potential rewards so they never miss earning opportunities.

---

## 🚀 Features

- Discover the latest TON ecosystem airdrops
- Instant alerts for new reward opportunities
- Simple Telegram commands for easy interaction
- AI-powered recommendations for airdrops
- Lightweight and easy to deploy

---

## 🧠 How It Works

1. Users start the Telegram bot.
2. The bot fetches active airdrops from the TON ecosystem.
3. Users can view opportunities using simple commands.
4. The bot sends alerts when new airdrops appear.

---

## 🤖 Bot Commands

/start - Start the bot and see introduction  
/airdrops - View latest TON ecosystem airdrops  
/alerts - Enable airdrop alerts  
/help - Show available commands

## 🛠 Tech Stack

- Python
- Telegram Bot API
- TON ecosystem integration
